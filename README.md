# 💎 Klinni IA - Gestão de Leads de Alto Padrão

Este é o repositório central do **Klinni IA**, um sistema focado em clínicas de odontologia e estética boutique.

## 🚀 Visão Geral
O sistema substitui falhas de softwares genéricos com:
- **Inteligência Geográfica:** Identificação de Leads High Ticket em Salvador.
- **Neuromarketing:** Sugestões de roteiros via Gemini para quebra de objeções.
- **Interface Soft & Clean:** Design minimalista para redução de estresse visual.

## 🛠️ Pilares do Projeto (Passo a Passo)

### 1. Categorização de Elite
- **HIGH TICKET:** Automático para +20 anos nos bairros: *Pituba, Horto, Graça, Vitória, Caminho das Árvores.*
- **TICKET PLUS:** Leads por indicação de pacientes atuais.
- **TICKET MÉDIO:** Demais perfis.

### 2. Stack Técnica
- **Frontend:** React.js (Interface) + Tailwind CSS (Estilo).
- **Backend:** Firebase (Banco de Dados e Login).
- **Cérebro:** Google Gemini API.

### 3. Regras de Acesso
- O primeiro usuário cadastrado é o **ADMIN**.
- Os demais são **CRC** (Comercial), com acesso restrito a vendas e sem acesso a logs de auditoria.
